# Wellness Check-In

A tranquil single-page Next.js experience that invites visitors to pause and notice how they feel right now. The interface offers a short mood-selector with reflective copy and gentle tips tailored to each choice.

## Tech Stack

- Next.js 14 with the App Router
- TypeScript
- Tailwind CSS for styling

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` and try the check-in flow. Production builds run with `npm run build` followed by `npm start`.

## Deploying

The project is ready for deployment on Vercel. Build locally before publishing to keep the production site healthy.
